## 📬 Project Inquiry & Email Setup

All inquiries submitted via our interactive project builder (`#contact`) are automatically dispatched directly to our core developer inbox at **`ortexdevs@gmail.com`**.

### How the Inquiry Setup Works:
1. **Direct Form Submission:** When a client selects their desired services, enters project scope details, and submits the form, our frontend sends a clean AJAX request to `https://formsubmit.co/ajax/ortexdevs@gmail.com`.
2. **One-Time Email Activation:**
   - The very first time a query is dispatched to `ortexdevs@gmail.com`, **FormSubmit.co** sends an activation link to the `ortexdevs@gmail.com` inbox.
   - Simply click **"Activate Form"** once inside that email, and all future client inquiries will instantly land in your inbox formatted cleanly with project details and multi-select service tags.
3. **Direct Email Fallback:** Prospective clients can also click `ortexdevs@gmail.com` anywhere on the site (`Contact Directory` or `Studio Footer`) to launch their mail client directly.

## 🚀 Local Development & Build

### Prerequisites
- Node.js (v18+)
- npm / pnpm / yarn

### Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/OrtexDevs/Landing-page.git
   cd Landing-page
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```
   The studio platform will launch cleanly at `http://localhost:3000`.

4. **Verify TypeScript & Build for Production:**
   ```bash
   npm run lint
   npm run build
   ```
